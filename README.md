## What is Pluto AI?

Pluto AI is a cybersecurity-only AI chatbot built for offensive security learners, CTF players, and penetration testers. It answers questions strictly within the cybersecurity and IT domain — anything outside that scope gets a **multilingual savage rejection** generated by the LLM itself.

Ask it about:
- Penetration testing techniques (SQLi, XSS, IDOR, SSRF, RCE)
- OWASP Top 10 vulnerabilities
- CTF tips and writeup walkthroughs
- Linux privilege escalation
- Nmap, Burp Suite, Metasploit usage
- Network security and forensics
- Malware analysis concepts
- OSINT and reconnaissance

## Features

### LLM Security Hardening
- Defensive system prompt architecture blocking **DAN-style jailbreaks**, role-override attempts (`"you are now..."`, `"ignore previous instructions"`), and system prompt extraction queries
- Strict **domain enforcement at the model level** — not just client-side filtering
- Off-topic queries trigger savage, multilingual single-line rejections generated by the LLM

### Multi-Session Chat
- Persistent chat history via `localStorage`
- Sessions **auto-titled** from the first user message
- Full support for session switching, deletion, and history replay across page reloads

### Hacker Aesthetic UI
- Dark terminal-style design with custom CSS variables
- Animated status indicators and pulsing status dot
- **Custom markdown renderer** built from scratch — no external libraries
  - Supports code blocks with syntax highlighting style
  - Headers, bold, inline code, lists
- Monospace typography (`Share Tech Mono`)
- Mobile-responsive layout

### Performance
- Zero backend — runs entirely in the browser
- Claude Sonnet API called directly from client
- Streaming-ready architecture

## Getting Started

### Prerequisites
- A **Groq API Key** — get one free at [console.anthropic.com](https://console.anthropic.com)

### Run Locally

```bash
git clone https://github.com/ROSHAN-Z89/PlutoAI.git
cd PlutoAI
# Open index.html in your browser
```

No build step, no npm install — pure HTML/CSS/JS.

### Set Your API Key

1. Open the app
2. Click **⚙ API Key** in the sidebar
3. Paste your Anthropic API key
4. Hit **Save Key** — stored locally in your browser

## Security Design

Pluto AI was built with LLM security as a first-class concern:

- **System prompt hardening** — the model is instructed to resist prompt injection, ignore override commands, and never reveal its instructions
- **Topic gating** — the model refuses to answer anything outside cybersecurity/IT, enforced at the LLM level
- **No server-side secrets** — API key is stored in the user's own browser, never sent to any third-party server
